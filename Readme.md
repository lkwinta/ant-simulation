# Ant Simulation

## Zależności
Aby stworzyć środowisko wirtualne z potrzebnymi zależnościami można użyć [uv](https://docs.astral.sh/uv/) i można je zainstalować za pomocą następującego polecenia:

```bash
uv sync
```

## Uruchomienie symulacji można przeprowadzić za pomocą polecenia:

```bash
cd simulation
python app.py
```
